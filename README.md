# Learning Elm + Parcel

Developing the [front-end test](https://gist.github.com/giulianovarriale/269f948dbf18573190ae59410be849b4) in [Elm](https://elm-lang.org) and using [Parcel](https://parceljs.org)

## Requirements

- Elm 0.19
- Node 8.9+

## Steps to run

1. `yarn install`
2. `yarn run start`
